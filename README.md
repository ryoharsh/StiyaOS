# 🚀 About

**StiyaOS** is an ambitious **Browser-Based Operating System** that recreates a complete desktop computing experience inside a web browser.

Unlike many browser desktop projects that simply display movable windows over a webpage, **StiyaOS is designed to behave like a real operating system**, featuring user authentication, desktop sessions, application lifecycle management, multitasking, and a scalable backend powered by the **MERN Stack**.

Our vision is to create an operating system that users can access from anywhere while preserving the familiarity, flexibility, and productivity of traditional desktop environments.

Instead of being "a website with windows," **StiyaOS aims to become a true Web Operating System.**

- MERN Backend

- Window Management
- Application Launcher
- Desktop Sessions
- State Management

The following applications are **currently under development** because they depend on the upcoming **Virtual File Management System**:

- 🎵 Media Player
- 🎬 Video Player
- 🖼 Photos App

Once the file system is completed, these applications will support persistent user files and behave like native desktop applications.

# 🎯 Project Vision

Our goal is **not** to build another web desktop clone.

We are building a complete **Web Operating System** capable of providing an experience comparable to traditional operating systems while remaining fully accessible through modern web browsers.

Future versions will support:

- User Workspaces
- Virtual File System
- Cloud Storage
- Advanced AI Assistant
- App Marketplace
- Browser-native Applications
- Cross-device Synchronization
- Offline Support
- Developer SDK
